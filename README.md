# Gestión de Tareas — Diplomado React

Aplicación web para gestionar tareas personales, desarrollada como proyecto final del Diplomado Full Stack con React.

## Tecnologías

- React 19 + TypeScript
- Material UI v9
- React Router v7
- Axios
- Zod
- Vite

## Funcionalidades

- Registro e inicio de sesión con JWT
- Listado paginado de tareas
- Crear, editar y eliminar tareas
- Cambiar estado entre Pendiente y Finalizada
- Rutas protegidas por autenticación
